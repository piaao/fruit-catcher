# 🍎 特殊水果事件系统 - 详细设计文档

## 一、设计概述

在现有6种普通水果 + 1种炸弹的基础上，引入 **6种特殊水果**。特殊水果不定期出现（概率触发），每种自带独特行为和效果，打破"吃→加分"的单调循环，增加每局的随机性和策略深度。

---

## 二、6种特殊水果详细设计

### ⭐ 1. 金色水果（Golden Fruit）

| 属性 | 值 |
|------|---|
| 外观 | 金色发光，脉冲光圈动画 |
| 生成概率 | 每15秒一次判定，8%概率 |
| 存在时间 | 4秒后自动消失（倒计时环） |
| 分数 | 50分 |
| 特殊行为 | **会跑！**每0.5秒随机改变方向一次，速度为普通水果的1.5倍 |
| 玩法 | 追踪型挑战，需要玩家快速反应去抓 |

**视觉表现：**
- 金色外发光（shadowBlur: 20, #ffd700）
- 缩放脉冲动画（sin波动 0.9~1.1）
- 被吃时：金色粒子爆炸 + "✨ +50" 浮动文字
- 即将消失时：闪烁加速（最后1秒每0.15秒闪一次）

**音效：**
- 生成时：上行琶音叮声（C5→E5→G5，短促）
- 被吃时：清脆金币声（高频三角波 + 泛音）
- 消失时：遗憾下行音（E4→C4）

---

### 🌈 2. 彩虹果（Rainbow Fruit）

| 属性 | 值 |
|------|---|
| 外观 | 持续彩虹色相旋转的球体 |
| 生成概率 | 每25秒一次判定，5%概率 |
| 存在时间 | 不会消失（持续在场直到被吃或掉出屏幕） |
| 分数 | 0分（自身不值分） |
| 特殊行为 | **吃到的瞬间触发"彩虹风暴"——全屏所有水果变为金色水果5秒** |
| 持续效果 | 5秒内所有水果都变金色外观、分值×3 |

**视觉表现：**
- 彩虹色相持续旋转（hue每帧+3）
- 外发光颜色跟随当前色相
- 触发彩虹风暴时：全屏彩虹光波从中心向外扩散（0.5秒）
- 风暴期间所有普通水果覆盖彩虹色 + 金色光环

**音效：**
- 生成时：空灵铃声（E5→G5→B5→E6，延音）
- 被吃/触发风暴时：宏大和弦 + 欢快旋律（C大调上行音阶）
- 风暴期间背景音微调（如有BGM则叠加和声层）

### 🧛 3. 腐坏果（Rotten Fruit）

| 属性 | 值 |
|------|---|
| 外观 | **伪装成随机一种普通水果**，但有微妙视觉区别 |
| 生成概率 | 每12秒一次判定，6%概率 |
| 存在时间 | 同普通水果（不特殊消失） |
| 分数 | -20分 |
| 特殊行为 | 伪装欺骗——外观95%像普通水果，有5%的视觉破绽 |
| 玩法 | 观察力挑战，仔细看才能分辨真假 |

**视觉破绽（5%差异，需要细心观察）：**
- 颜色略微发暗/发灰（亮度降低15%）
- 偶尔冒出小绿气泡（每1.5秒冒一个，很微小）
- emoji上方有极小的虫子图标（🐛，4px大小）
- 飞行轨迹偶尔轻微抖动（sin波动 ±2px）

**难度递进：**
| 关卡 | 破绽明显程度 |
|------|------------|
| 1-2关 | 颜色暗20% + 虫子图标6px + 抖动明显 |
| 3关 | 颜色暗15% + 虫子图标5px + 抖动中等 |
| 4-5关 | 颜色暗10% + 虫子图标4px + 抖动微弱 |

**视觉表现（被吃时）：**
- 绿色毒液飞溅粒子
- 屏幕边缘短暂绿色晕染
- "🤢 -20" 浮动文字

**音效：**
- 被吃时：恶心音效（低频不和谐音 + "噗"声）

---

### 🍬 4. 冰冻果（Ice Fruit）

| 属性 | 值 |
|------|---|
| 外观 | 冰蓝色半透明球体，内有雪花粒子 |
| 生成概率 | 每20秒一次判定，6%概率 |
| 存在时间 | 同普通水果 |
| 分数 | 10分 |
| 特殊行为 | **吃到后所有水果冻结3秒（包括炸弹）** |
| 冻结效果 | 所有投射物速度降至10%，可从容选择吃哪些 |

**视觉表现：**
- 冰蓝色半透明（rgba(150, 220, 255, 0.7)）
- 内部雪花粒子旋转
- 外层冰晶光圈
- 触发冻结时：全屏冰霜扩散效果（蓝白色从角色向外扩散）
- 冻结期间所有水果覆盖冰层纹理（半透明白色 + 冰裂纹）
- 冻结结束时：碎冰粒子效果

**音效：**
- 生成时：清脆冰裂声（高频噪声短促）
- 被吃时：冰冻扩散音（白噪声 + 下行滑音）
- 冻结结束：碎裂声（多个短促click）

---

### 💫 5. 分裂果（Split Fruit）

| 属性 | 值 |
|------|---|
| 外观 | 较大的紫红色球体，内部有分隔线 |
| 生成概率 | 每20秒一次判定，5%概率 |
| 存在时间 | 同普通水果 |
| 分数 | 8分（本体）+ 每个小碎片5分 |
| 特殊行为 | **吃到后分裂成3个小碎片向四周飞散** |
| 玩法 | 技巧挑战——吃一个触发三个，需要快速连续接住碎片 |

**视觉表现：**
- 大号紫红色球体（radius 30），有明显的中线
- 被吃时：分裂动画——本体裂开成3个小球
- 3个碎片颜色略有深浅差异（暗示可吃）
- 碎片有拖尾效果（淡紫色）
- 碎片飞行2秒后自动消失（渐隐）

**音效：**
- 生成时：充满张力的"嗡"声（低频共振）
- 被吃（分裂时）：爆裂声 + 3声快速连续弹跳音
- 碎片被接住：清脆弹跳声（每个音调略不同 C5/E5/G5）

---

### 🎯 6. 连锁果（Chain Fruit）

| 属性 | 值 |
|------|---|
| 外观 | 橙黄色球体，外围有闪电符号 |
| 生成概率 | 每25秒一次判定，4%概率 |
| 存在时间 | 同普通水果 |
| 分数 | 15分（本体）+ 连锁额外分 |
| 特殊行为 | **吃到后触发"水果链"，自动吸附150px范围内所有水果飞向玩家** |
| 连锁效果 | 被吸附的水果也算"吃到"，正常加分 |
| 最大连锁数 | 最多连锁6个水果（防止数值膨胀） |

**视觉表现：**
- 橙黄色球体 + 闪电图标
- 被吃时：闪电从角色向外扩散（射线效果）
- 被连锁的水果之间有闪电连线
- 被吸附的水果飞向玩家时留下光轨
- 连锁结束后显示 "⚡ 连锁 ×N" 大字

**音效：**
- 生成时：电流滋滋声
- 被吃时：雷电音效（白噪声 + 低频轰鸣）
- 每个连锁水果到达时：叠加一声清脆铃声（音调递增 C→D→E→F→G→A）

## 三、生成系统设计

### 独立计时器

每种特殊水果有**独立的定时判定器**，互不干扰：

```
goldenTimer:  15000ms → 8% 概率
rainbowTimer: 25000ms → 5% 概率
rottenTimer:  12000ms → 6% 概率
iceTimer:     20000ms → 6% 概率
splitTimer:   20000ms → 5% 概率
chainTimer:   25000ms → 4% 概率
```

### 生成规则
- 计时器在 `state === 'playing'` 时才递增
- 判定成功后计时器重置，重新开始倒计时
- 同一时刻场上最多存在 **2个** 特殊水果（避免视觉混乱）
- 金色水果即将消失时（最后1秒）有30%概率再生成一个新金色水果（引诱玩家跑过去）

### 难度缩放

| 关卡 | 特殊水果总生成率加成 |
|------|---------------------|
| 1关 | ×0.6（新手期少一些） |
| 2关 | ×0.8 |
| 3关 | ×1.0（标准） |
| 4关 | ×1.2 |
| 5关 | ×1.5（高手期更多刺激） |

## 四、状态效果管理

### 全局效果栈

```javascript
const activeEffects = {
  rainbow: { active: false, timer: 0, duration: 5000 },  // 彩虹风暴
  freeze:  { active: false, timer: 0, duration: 3000 },  // 冰冻效果
};
```

### 效果叠加规则
- **彩虹风暴 + 冰冻**：可以同时存在。冰冻的水果也是彩虹色的金色水果
- **连锁果在冰冻期间触发**：被吸附的水果也是冻结状态，慢慢飞过来（视觉效果很诡异很有趣）
- **彩虹风暴 + 连锁果**：连锁的每个水果都是×3分数（超高收益！）

### 效果UI提示
- 屏幕右上角（HUD下方）显示当前激活效果的图标 + 剩余时间条
- 例如：`🌈 3.2s` `❄️ 1.5s`

## 六、代码架构设计

### 新增数据结构

```javascript
// 特殊水果类型
const SPECIALS = {
  golden:  { name:'金色水果', ... },
  rainbow: { name:'彩虹果', ... },
  rotten:  { name:'腐坏果', ... },
  ice:     { name:'冰冻果', ... },
  split:   { name:'分裂果', ... },
  chain:   { name:'连锁果', ... },
};

// 投射物扩展字段
projectile.special = 'golden' | 'rainbow' | 'rotten' | 'ice' | 'split' | 'chain' | null
projectile.specialTimer = 4000  // 金色水果的剩余存活时间
projectile.lifetime = 2.0      // 分裂碎片的剩余飞行时间
projectile.chained = true       // 是否被连锁吸附
```

### 修改点清单

1. **spawnProjectile()** — 加入特殊水果生成判定
2. **render()** — 特殊水果的专属渲染逻辑
3. **主循环** — 特殊水果定时器递减 + 金色水果移动方向变化 + 冻结速度修改
4. **碰撞处理** — 每种特殊水果的吃到效果
5. **HUD** — 效果图标 + 剩余时间条
6. **成就系统** — 新增10个成就 + 统计追踪
7. **音效系统** — 6种特殊水果的生成/吃到/消失音效
8. **endRound()** — 效果全部清除

### 渲染优先级（从底到顶）
```
背景 → 冻结效果滤镜 → 普通投射物 → 特殊投射物 → 连锁闪电线 → 
分裂碎片拖尾 → 金色水果光圈 → 粒子效果 → 浮动文字 → 
倒计时数字 → HUD → 成就提示 → 效果状态栏
```

## 七、实现顺序建议

| 阶段 | 内容 | 预计工作量 |
|------|------|-----------|
| Phase 1 | 金色水果 + 冰冻果 + 效果系统框架 | 基础，其他特殊水果依赖此框架 |

| Phase 4 | 新成就 + 平衡性调整 + 测试 | 收尾 |
