# 🌍 水果吃货 - 50关主题关卡设计文档

## 一、设计概述

将游戏从5关扩展至**50关**，按水果分类分为**5大主题**，每主题10关。每个主题拥有专属的：
- **主题人物**（玩家角色外观）
- **游戏背景**（3-5张随机切换）
- **关卡音乐**（3-5首随机播放，Web Audio API 程序化合成）
- **主题水果**（该主题独有的水果池 + 音效）

### 主题总览

| 主题 | 关卡范围 | 主题名 | 色调 | 难度区间 |
|------|---------|--------|------|---------|
| 1 | 1-10 | 🍑 核果园 | 粉紫暖色 | 入门→初级 |
| 2 | 11-20 | 🍊 柑橘岛 | 金橙阳光色 | 初级→中级 |
| 3 | 21-30 | 🫐 浆果谷 | 紫蓝冷色 | 中级→高级 |
| 4 | 31-40 | 🥭 热带雨林 | 翠绿热带色 | 高级→专家 |
| 5 | 41-50 | 🍎 仁果殿 | 深红金色 | 专家→大师 |

## 二、通用关卡难度曲线

50关难度参数按指数曲线递增，确保平滑过渡：

```
baseSpeed:     100 + level * 4          (100 → 300)
spawnInterval: 1800 - level * 26        (1774 → 500ms)
bombChance:    0.06 + level * 0.0036    (0.064 → 0.24)
target:        50 + level * 16          (66 → 850)
roundTime:     45秒（固定，高关卡可适当延长至50秒）
```

### 难度分级标签

| 关卡区间 | 难度标签 | 特殊机制 |
|---------|---------|---------|
| 1-5 | 新手教程 | 低速、少炸弹、道具掉落率×1.5 |
| 6-10 | 初级挑战 | 引入主题机制 |
| 11-15 | 进阶挑战 | 速度提升明显 |
| 16-20 | 中级考核 | 炸弹频率增加 |
| 21-25 | 高级入门 | 引入新水果特性 |
| 26-30 | 高级挑战 | 水果速度差异化加大 |
| 31-35 | 专家入门 | 密集弹幕 |
| 36-40 | 专家考核 | 炸弹比例提升 |
| 41-45 | 大师入门 | 极高速度+密集 |
| 46-50 | 终极挑战 | 全属性拉满 |

    endLevel: 9,

    bgmSeeds: [ /* 音乐种子数组 */ ],
  },
  // ... 共5个主题

// 获取当前主题
function getCurrentTheme() {
  return THEMES.find(t => level >= t.startLevel && level <= t.endLevel);

### BGM 系统设计

采用 Web Audio API 程序化生成，每个主题有多首 BGM 种子：

```javascript
// BGM 系统
const bgmSystem = {
  playing: false,
  currentSeed: null,
  audioBuffer: null,
  
  // 根据主题和关卡选择 BGM
  playForTheme(themeId) {
    const theme = getCurrentTheme();
    const seed = theme.bgmSeeds[Math.floor(Math.random() * theme.bgmSeeds.length)];
    this.playBGM(seed);
  },
  
  // 程序化生成BGM循环
  playBGM(seed) { /* ... */ },
  stop() { /* ... */ },
  setVolume(v) { /* ... */ },
};
```

每首 BGM 约 8-16 小节循环，用 OscillatorNode + GainNode 编排旋律。
