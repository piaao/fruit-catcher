# 🌍 水果吃货 - 50关主题关卡设计文档

## 一、设计概述

将游戏从5关扩展至**50关**，按水果分类分为**5大主题**，每主题10关。每个主题拥有专属的：
- **主题人物**（玩家角色外观）
- **游戏背景**（3-5张随机切换）
- **关卡音乐**（3-5首随机播放，Web Audio API 程序化合成）
- **主题水果**（该主题独有的水果池 + 音效）

### 通用美术规格

#### 游戏画布

- **分辨率**：全屏自适应，`window.innerWidth × window.innerHeight`
- **常见参考尺寸**：1920×1080（桌面）、1366×768（笔记本）
- 背景图需支持动态缩放，以 `cover` 模式铺满画布

#### 角色图片规格

| 参数 | 规格 |
|------|------|
| **视角** | 正面45°俯视（3/4视角），角色面朝屏幕偏左上方 |
| **尺寸** | 128×128px（逻辑像素），@2x导出为 256×256px |
| **格式** | PNG（带Alpha透明通道） |
| **动效** | 需要 sprite sheet 动效图 |
| **动效帧数** | idle（待机）4帧循环，eat（吃水果）6帧一次性 |
| **动效排列** | 横向排列，每帧128×128px |
| **总图尺寸** | idle: 512×128px，eat: 768×128px |
| **总逻辑碰撞范围** | 头部半径 24px（含头部半径+4px容差） |

> **说明**：当前版本使用 Canvas 程序化绘制角色（圆形身体+圆形头部+装饰物），后续可替换为 PNG sprite sheet 实现更精致的拟人化角色。本规格为 PNG 替换阶段预留。

#### 背景图片规格

| 参数 | 规格 |
|------|------|
| **基准尺寸** | 1920×1080px（横向16:9），@1x |
| **备选尺寸** | 1920×1200px（横向16:10），适配部分笔记本 |
| **格式** | JPG（有损压缩，文件小）或 WebP（推荐） |
| **色调** | 各主题有专属色调（见下文），背景整体偏暗以衬托前景元素 |
| **构图** | 主体居中偏下，上下留出 HUD 叠加空间（上方60px、下方40px），边缘做渐暗处理（vignette） |
| **缩放策略** | CSS `cover` 模式铺满，超出部分裁切，保证无黑边 |

> **注意**：背景尺寸已从旧版 800×600 升级为全屏规格。旧提示词中的 `1920x1080` 仅作历史参考，实际生成时使用 1920×1080。

## 二、通用关卡难度曲线

50关难度参数按指数曲线递增，确保平滑过渡：

```
baseSpeed:     100 + level * 4          (100 → 300)
spawnInterval: 1800 - level * 26        (1774 → 500ms)
bombChance:    0.06 + level * 0.0036    (0.064 → 0.24)
target:        50 + level * 16          (66 → 850)
roundTime:     45秒（固定，高关卡可适当延长至50秒）
```

### 难度分级标签

| 关卡区间 | 难度标签 | 特殊机制 |
|---------|---------|---------|
| 1-5 | 新手教程 | 低速、少炸弹、道具掉落率×1.5 |
| 6-10 | 初级挑战 | 引入主题机制 |
| 11-15 | 进阶挑战 | 速度提升明显 |
| 16-20 | 中级考核 | 炸弹频率增加 |
| 21-25 | 高级入门 | 引入新水果特性 |
| 26-30 | 高级挑战 | 水果速度差异化加大 |
| 31-35 | 专家入门 | 密集弹幕 |
| 36-40 | 专家考核 | 炸弹比例提升 |
| 41-45 | 大师入门 | 极高速度+密集 |
| 46-50 | 终极挑战 | 全属性拉满 |

```javascript
// 绘制时使用 cover 模式铺满
function drawBgImage(img) {
  const imgRatio = img.width / img.height;
  const canvasRatio = W / H;
  let sw, sh, sx, sy;
  if (canvasRatio > imgRatio) {
    sw = img.width; sh = img.width / canvasRatio;
    sx = 0; sy = (img.height - sh) / 2;
  } else {
    sh = img.height; sw = img.height * canvasRatio;
    sx = (img.width - sw) / 2; sy = 0;
  }
  ctx.drawImage(img, sx, sy, sw, sh, 0, 0, W, H);
}
```
